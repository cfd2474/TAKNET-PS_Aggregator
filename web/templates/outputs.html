REST API (v2)

Airplanes.live–compatible · public, no auth

This aggregator exposes an ADS-B REST API at /v2/ with the same endpoint structure and response format as airplanes.live. Use it to query merged aircraft (local + ADSBHub) by hex, callsign, point/radius, and more.

Example: curl "https://adsb.tak-solutions.com/v2/point/33.8753/-117.5664/50" — aircraft within 50 nm of a point.

Endpoint Method Description
/v2/hex/<hex>GETAircraft with exact match on one or more Mode S hex IDs (comma-separated, limited to 1000)
/v2/callsign/<callsign>GETAircraft with exact match on one or more callsigns
/v2/reg/<reg>GETAircraft with exact match on one or more registrations
/v2/type/<type>GETAircraft with specified ICAO type code (e.g. A321, B738)
/v2/squawk/<squawk>GETAircraft squawking the specified value
/v2/milGETAircraft tagged as military
/v2/laddGETAircraft tagged as LADD
/v2/piaGETAircraft tagged as PIA
/v2/point/<lat>/<lon>/<radius_nm>GETAircraft within radius (nautical miles) of a point; radius up to 250 nm
/v2/allGETAll currently tracked aircraft
/v2/healthGETService health check

Responses use the same JSON envelope and field names as airplanes.live. For data field descriptions, see airplanes.live API guide and data field docs. Use a reasonable request rate (e.g. 1 request per second) when polling.
