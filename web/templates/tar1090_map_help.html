This page summarizes how to use the tar1090-style map (filters, keyboard shortcuts, URL parameters). Content is adapted from the public ADS-B Exchange map help for convenience; the authoritative reference is linked below.

tar1090 map key — official reference (ADS-B Exchange) ↗

Search box

Universal search finds ICAO (hex), registration, and aircraft type.

Filters

Open Filters in the right sidebar to filter by altitude range, callsign, squawk, type, or type description.

  • . is a placeholder for any character.
  • Combine patterns with |.
  • Do not add spaces — filtering will not work.

Type description examples

  • Helicopters: H..
  • Landplanes, 2 jet engines: L2J
  • Landplanes, any pistons: L.P
  • Helicopters, any turbines: H.T
  • All turboprop (incl. turbine helicopters): ..T
  • Four engines: .4.
  • 2, 3, or 4 engines: .2.|.3.|.4.

ICAO type code examples

  • B737 family: B73.
  • A320 family: A32.
  • B737-900 / Max 9: B739|B39M
  • 737 incl. Max: B73.|B3.M
  • B737 / A320: B73.|B3.M|A32.|A2.N
  • Exclude a type: ^(?!A320)
  • Exclude multiple: ^(?!(A32.|B73.))

Toolbar buttons & keyboard

Press the letter on the keyboard or click the button.

UMilitary / Interesting TTracks on for all aircraft OLabel extensions BMap brightness MMulti-select IIsolate selected aircraft FFollow plane (map moves) HHome / reset map LLabels KTrack labels PPersistence (planes stay after signal lost) RFollow random plane

Top-left: icon size. Lower-right +/- or scroll wheel: zoom.

Map layers & gear

Map layers — basemaps, charts, overlays (e.g. weather), your location dot, range rings.
Gear — display options including units of measure.

Aircraft list

Click an aircraft in the list as on the map. The list is limited to about 100 aircraft; zoom in if you do not see a plane you see on the map.

Colors & traces

Altitude is shown as a gradient (orange → purple → red). Gray = on the ground in ground mode.
Shift+L — toggle estimated last leg vs full trace.
Shift+S — temporarily hide navigation buttons.

Location & range rings

The map may use browser geolocation if allowed. Set a fixed site position with URL parameters, e.g. ?SiteLat=XX.XXX&SiteLon=-XX.XXX (bookmark for repeat visits). Range rings are toggled in map layers; ring distances are fixed by the app.

Mobile

Layout is adjusted for small screens; increase icon size for easier taps. Mobile may cap visible aircraft (~100), loading low-to-high altitude when zoomed out — zoom in for full local traffic.

URL parameters

First query uses ?, additional use &. Parameters are generally case-insensitive. Some are flags (no value); some need values (default if omitted).

Common

  • icao=A1B2C3 — center, select, isolate
  • icaoFilter=hex1,hex2,hex3 — only listed hexes
  • hideSidebar, hideButtons
  • tempTrails=<seconds>
  • largeMode=<1–4> — icon size cycle
  • sidebarWidth=<px>
  • mil — military/interesting
  • kiosk — minimal UI, icon size 2
  • zoom=<1–20>
  • lat= & lon= — center
  • airport=KJFK (etc.) — center on airport
  • replay — replay toolbar

Render / display

  • mapOrientation=0–360
  • monochromeMarkers=RRGGBB, monochromeTracks=RRGGBB
  • outlineWidth, outlineColor (WebGL off)
  • scale, iconScale, labelScale
  • mapDim, mapContrast

Troubleshooting

  • reset — reset page settings
  • showerrors — show errors on screen

Heatmaps

Example: ?heatmap=200000 with optional heatDuration, heatEnd, heatRadius, heatAlpha, heatManualRedraw, realHeat, heatBlur, heatWeight. Heavy — use a powerful machine or lower dot count.
