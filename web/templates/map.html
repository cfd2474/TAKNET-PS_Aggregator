tar1090 map key

How to use the tar1090 map: search, filters, keyboard shortcuts, and URL parameters.

Search box

Universal search finds ICAO (hex), registration, and aircraft type.

Filters

Open Filters in the right sidebar for altitude, callsign, squawk, type, or type description.

Filter tab in the sidebar — no spaces in patterns; use | to combine; . matches any character.
  • . = any character
  • Combine with |
  • No spaces in patterns

Type description examples

  • Helicopters: H..
  • Landplanes, 2 jets: L2J
  • Landplanes, any pistons: L.P
  • Helicopters, any turbines: H.T
  • All turboprop: ..T
  • Four engines: .4.
  • 2 / 3 / 4 engines: .2.|.3.|.4.

ICAO type code examples

  • B737: B73. · A320: A32.
  • B739 / Max 9: B739|B39M
  • 737 incl. Max: B73.|B3.M
  • B737 / A320 mix: B73.|B3.M|A32.|A2.N
  • Exclude type: ^(?!A320)
  • Exclude several: ^(?!(A32.|B73.))

Buttons

Each toolbar control has a letter — click or press that key on the keyboard.

U Military / interesting
H Home/Reset Map
T Tracks on for all aircraft
L Labels
O Label extensions
K Track labels
B Map brightness
M Multi-select
P Persistence — planes stay after signal lost
I Isolate selected aircraft
R Follow random plane
F Follow plane (map moves)
Top left — icon size Lower right + / − or scroll — zoom

Map layers

Basemaps, charts, overlays (e.g. weather), your location dot, and range rings.

Gear icon

Display options here, including units of measure.

Aircraft list

Click a row like clicking the map. The list shows about 100 aircraft — zoom in if you do not see a plane that is on the map.

Airplane colors

Altitude gradient: orange → purple → red. Gray = on the ground in ground mode.

Shift+L — estimated last leg vs full trace · Shift+S — hide navigation buttons.

Location & range rings

The map will attempt to detect your location automatically as long as you allow your browser to provide the info.

Range Rings can be turned on and off in the Map Layers Icon, there is no way to adjust the ranges displayed.

Mobile

Layout fits small screens; increase icon size for easier use. Some builds cap at ~100 planes when zoomed out (loaded low→high altitude) — zoom in to see all nearby traffic.

URL parameters

First query uses ?, then &. Usually case-insensitive. Some flags need no value.

Common

  • icao= — center, select, isolate
  • icaoFilter=hex1,hex2
  • hideSidebar, hideButtons
  • tempTrails=<sec>, largeMode=1–4, sidebarWidth=
  • mil, kiosk, zoom=, lat= & lon=, airport=, replay

Display

  • mapOrientation=, monochromeMarkers=, monochromeTracks=
  • outlineWidth, outlineColor (no WebGL)
  • scale, iconScale, labelScale, mapDim, mapContrast

Troubleshooting

  • reset · showerrors

Heatmaps

heatmap=<max dots> — optional heatDuration, heatEnd, heatRadius, heatAlpha, heatManualRedraw, realHeat, heatBlur, heatWeight. Heavy on the CPU; use lower counts on slower devices.
